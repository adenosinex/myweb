📌 核心分类法则 (最高控制在60个词以内)

1. 四维强制打标法

每张卡片标签数严格控制在 3 ~ 5 个，必须从以下维度拼图：

  • 时长媒介：短视频 / 长片 / 合集精剪
  • 调性来源：素人真实 / 专业厂牌 / 网黄福利
  • 人身体态：少女 / 少妇 / 饱满 / 纤细 / 微胖
  • 场景行为：裸舞 / 剧情 / POV视角 / 户外 / 车内

2. 严禁同义词灾难

绝对不允许同时出现“丰满”和“饱满”，或“年轻”和“少女”。选定一个主干词后，永久沿用。

3. 拒绝正文污染

标签是用来聚类检索的，不是用来写描述的。具体的衣服颜色、对白、演员名字，请写在底部的 Markdown 详情正文里！
